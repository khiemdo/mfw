Xbus message reading utility functions.

Xbus messages use big-endian representations for multibyte data. These functions help in reading data from an Xbus message and converting it to a native type.

The functions are intended to scan over a message as follows:

void readValues(XbusMessage const* message)
{
uint16_t v1;
uint8_t v2;
uint32_t v3;
uint8_t* dptr = message->data;
dptr = XbusUtility_readU16(&v1, dptr);
dptr = XbusUtility_readU8(&v2, dptr);
dptr = XbusUtility_readU32(&v3, dptr);
}
