uint32_t hal_timer_getTimeout ( uint32_t  ui32_timeoutMs)

Initializes and starts a single timer by returning a timeout value.

The user gives the timeout value that is the relative future time when the timer expires. See hal_timer_isTimedOut for an example usage of this API.

Parameters
ui32_timeoutMsTime to expire (in milliseconds).
Returns
Timer instance as reference to the requested timeout. The returned value can be passed to hal_timer_isTimedOut to check if the timrout occurred or not.

bool hal_timer_init ( void  )

Initialize the HAL timer module.

A call to this function is required prior to use any timers! After a call to this function, a HW timer is initialized to work as a time base with a resolution of 1 millisecond. Following operations are performed by this function:

  • Initialize a MCU internal timer with a resolution of 1ms.
  • Enable timer interrupt.
Returns
TRUE if initialized successfully. FALSE otherwise.
