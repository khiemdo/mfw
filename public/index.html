Documentation

The API documentation of the repository is managed and maintained using Doxygen.

The Doxygen configuration file (doxyfile-embMasterDude) can be found in /doc folder of the repository. Doxygen generates a new folder /doc/output that holds the documentation output. This folder is ignored by Git as the documentation can and shall be generated upon request any time. Furthermore, Doxygen generates a lot of files and many changes within these files to create the documentation, that is not worth versioning as it creates more extra work than benefit.
