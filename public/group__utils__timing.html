uint32_t timing_getTimeout ( uint32_t  ui32_timeoutMs)

Initializes and starts a single timer by returning a timeout value.

The user gives the timeout value that is the relative future time when the timer expires. See timing_isTimedOut for an example usage of this API.

Parameters
ui32_timeoutMsTime to expire (in milliseconds).
Returns
Timer instance as reference to the requested timeout. The returned value can be passed to timing_isTimedOut to check if the timrout occurred or not.
