bool hal_mcu_init ( void  )

Initialize the microcontroller driver module.

A call to this function is required prior to any HAL operations (e.g. to use the HAL SPI driver for SPI transmissions)! Following operations are performed by this function:

  • Initialize the system clock module (default: 120MHz system clock).
  • Enable processor interrupts.
Returns
TRUE if initialized successfully. FALSE otherwise.
