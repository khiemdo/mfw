59 # Documentation {#docu}
60 
61 The API documentation of the repository is managed and maintained using [Doxygen].
62 
63 The Doxygen configuration file (`doxyfile-embMasterDude`) can be found in `/doc` folder of the repository.
64 Doxygen generates a new folder `/doc/output` that holds the documentation output. This folder is ignored by Git as the documentation can and shall be generated upon request any time.
65 Furthermore, Doxygen generates a lot of files and many changes within these files to create the documentation, that is not worth versioning as it creates more
66 extra work than benefit.
67 
68 [Doxygen]: http://www.stack.nl/~dimitri/doxygen/index.html "Doxygen"
