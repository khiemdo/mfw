void ads1115_run ( s_ads1115_ctx_t *  ads_ctx)

Handle the ADS-sensor state.

Parameters
ads_ctxDefines the context to the ADS-instance that should be handled.
Returns
None.

Update the busmanager

Set the write/read-ctx according to the ads-ctx.

Perform actions depending on current state of instance IDLE: Send measurement instruction -> PENDING PENDING: Wait for callback -> MEASURING MEASURING: Wait for measurement timer to run out -> READY READY: Read measured data -> PENDING PENDING: Wait for callback -> IDLE (If timeout while PENDING -> TIMEOUT)
