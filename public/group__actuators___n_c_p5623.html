void NCP5623_rainbow ( uint16_t  speed)

Generates a rainbow.

This function continuously changes the data written to the LED controller in a way, that all possible colours are generated. It automatically ramps up and down the PWM values with a given speed per step.

void NCP5623_run ( )

Run function for measurement framework.

This function is repeatedly called by the measurement framework. It realizes a state machine for the LED controller. If the LED controller is idle, it calls the set function. If a transmission is pending, it does nothing. If the controller is in the progress of dimming up or down, it checks if the controller has finished. If not, it also exits, otherwise state is changed to idle.

Returns
None.

void NCP5623_set ( )

Writes the new values stored in memory to the LED controller.

This function implements the logic for writing new values to the LED controller. It checks if new values are present and if so, writes them to the LED controller. It makes sure, that all changed values are written in a row, before the next new data is written. The function also start a timer for the dimming time, if dimming is wanted and determines the dimming direction and writes the values accordingly.

Returns
None.

void NCP5623_shutdown ( )

Shuts the LED controller down.

This function shuts down all output and essentially deactivates the LEDs. It is not a reset. All values stored in the LED controllers registers are still present. As soon as a value is written into one of the registers, all outputs are activated again.

Returns
None.
