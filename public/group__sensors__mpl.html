bool MPL_get ( Barometer *const  p_baro_msg)

gets the current barometer values and writes them to protobuf msg.

Be aware, that this function has to be used in combination with the *_run() function, as that function handles the actual sensor readout. This function only copies local new values to msg.

Parameters
p_baro_msgPointer to msg struct.
Returns
TRUE if new sensor data is written into the message. FALSE otherwise.
