void hal_pps_isr ( void  )
inline

Interrupt service routine for timestamping.

Is executed immediately on interrupt for shortest possible delay. Declared inline to avoid unnecessary jumps in code. Compiler should be called with highest possible -O option. The function reads the current counter value of the assigned timer and resets it. The PPS counter is increased. Both values are stored in internal variables.

Returns
None.
