bool SHT25_checkCRC ( SHT25_DATA_t  ui16_data,
SHT25_CRC_t  ui8_crc 
)

Checks the CRC8 checksum of the measurement.

This function can be used for checking the validity of the measurement data. It calculates the CRC8 checksum with the polynomial 0x131 and compares it with the checksum received from the sensor.

Parameters
ui16_dataRaw date received from the sensor.
ui8_crcCRC8 value received from the sensor.
Returns
TRUE if checksum matches. FALSE otherwise.
