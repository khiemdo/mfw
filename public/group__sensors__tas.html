bool tube_angle_sens_init ( uint32_t  ui32_spiClock)

Initialize the tube-angle-sensor.

Please note that the module internally is disabled by default after initialization. To enable this module a call to tube_angle_sensor_setConf() is required with parameter "activated" set to true.

Parameters
ui32_spiClockDefines the spi-clk speed to be intialized.
Returns
Returns pointer to context variable if tube-angle-sensor is initialized successfully. NULL otherwise.

Initialize our context struct and init SPI port.

Configure the SPI port

initialize GPIO module with two GPIO-pins for CS0 and CS1 (elevation, rotation)

Set both CS to idle

bool tube_angle_sensor_get ( AngleEncoder *const  p_ae_el_msg,
AngleEncoder *const  p_ae_ro_msg 
)

Gets the current elevation and rotation of the tube-angle-sensor and writes the values to the provided protobuf tas-msg.

Be aware, that this function has to be used in combination with the *_run() function, as that function handles the actual SPI transmission. This function only copies local new values to msg.

Returns
TRUE if new sensor data is written into the message. FALSE otherwise.

Proceed only if module is enabled and a valid pointer is given.

get elevation value
